# Dataset

## SMS Spam Collection Dataset

This project uses the SMS Spam Collection Dataset for spam message classification.

### Features
- label (spam / ham)
- message

### Dataset Size
- Total Messages: 5572
- Spam Messages: 747
- Ham Messages: 4825

### Source
https://archive.ics.uci.edu/ml/datasets/sms+spam+collection

> This dataset is used for educational and research purposes.
